## 🚀 TodoPro – Smart Task Management Web App

TodoPro is a **full-stack task management web application** built to help users organize, track, and complete daily tasks efficiently.

It features secure authentication, real-time analytics, task prioritization, productivity tracking, and export functionality — all wrapped in a clean responsive UI.

---

### ✨ Key Features

* 🔐 **Secure Authentication** – User-specific tasks with JWT-based login
* 📝 **Task Management** – Add, update, complete, and delete tasks
* 📅 **Due Dates & Priority Levels** – Low, Medium, High
* 📊 **Interactive Dashboard** – Real-time stats & charts
* 📈 **Productivity Tracking** – Weekly productivity visualization
* 📄 **Export Options** – Download tasks as **PDF** or **CSV**
* 💡 **Responsive Design** – Works seamlessly on all devices

---

### 🛠️ Tech Stack

**Frontend**

* HTML5, CSS3, JavaScript
* Chart.js
* jsPDF

**Backend**

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT Authentication

**Deployment**

* Frontend: **GitHub Pages**
* Backend: **Render**
* Database: **MongoDB Atlas**
